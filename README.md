# AsyncRecon Framework

إطار عمل متقدم للفحص الشبكي غير المتزامن (Asynchronous Port Scanner). تم تصميم هذه الأداة للباحثين الأمنيين الذين يحتاجون إلى سرعة فائقة في الاستطلاع (Reconnaissance) واكتشاف الخدمات المفتوحة في بيئات الشبكات المعقدة.

---

## 🚀 لماذا AsyncRecon؟

في عمليات اختبار الاختراق والـ Bug Bounty، الوقت هو أثمن الموارد. تستخدم `AsyncRecon` محرك `asyncio` المدعوم بـ `uvloop` (أسرع محرك حلقات أحداث لـ Python)، مما يسمح بفحص آلاف المنافذ في ثوانٍ معدودة بكفاءة عالية واستهلاك منخفض للموارد.

### الميزات التقنية:
* **محرك غير متزامن (High-Performance Engine):** يعتمد على `uvloop` لتقليص زمن الاستجابة (Latency) إلى أقصى حد.
* **إدارة ذكية للموارد:** ترفع الأداة حدود النظام تلقائياً (File Descriptors) لمنع فشل الاتصال أثناء عمليات المسح المكثفة.
* **واجهة مستخدم تفاعلية (TUI):** مدعومة بمكتبة `rich` لعرض تفاصيل الفحص الحية، أشرطة التقدم، وجداول النتائج.
* **Banner Grabbing:** استخراج فوري للترويسات للتعرف على الخدمات والنسخ العاملة على المنافذ المكتشفة.

---

## 🛠 المتطلبات (Requirements)
* Python 3.8 أو أحدث.
* نظام تشغيل UNIX-like (Linux/macOS) للحصول على أفضل أداء وتوافق مع `uvloop`.

## 📦 التثبيت والإعداد (Setup & Installation)

للبدء، اتبع الخطوات التالية بالترتيب لضمان إعداد بيئة عمل نظيفة ومستقرة:

```bash
# 1. استنساخ المستودع (أو التوجه لمجلد المشروع)
git clone [https://github.com/yourusername/AsyncRecon.git](https://github.com/yourusername/AsyncRecon.git)
cd AsyncRecon

# 2. إنشاء بيئة افتراضية (Virtual Environment) لعزل الاعتماديات
python3 -m venv recon_env

# 3. تفعيل البيئة
source recon_env/bin/activate

# 4. تثبيت المكتبات والمتطلبات
pip install -r requirements.txt

# 5. منح صلاحيات التنفيذ للسكربت
chmod +x async_recon.py
💻 الاستخدام (Usage)بعد تفعيل البيئة، يمكنك تشغيل الأداة بسهولة. الصيغة العامة للأمر هي:Bashpython async_recon.py -t <TARGET_IP> [خيارات إضافية]
خيارات التحكم المتاحة:العلم (Flag)الوصفالافتراضي-t, --targetعنوان IP المستهدف (إجباري).--c, --concurrencyعدد الاتصالات المتزامنة (Concurrency).1000-to, --timeoutمهلة الانتظار لكل منفذ بالثواني.1.5-s, --startمنفذ البداية لنطاق المسح.1-e, --endمنفذ النهاية لنطاق المسح.65535أمثلة عملية:فحص سريع للنطاق الافتراضي:Bashpython async_recon.py -t 192.168.1.1
فحص مكثف وشامل (جميع المنافذ) مع رفع التزامن:Bashpython async_recon.py -t 10.0.0.5 -c 5000 -s 1 -e 65535
فحص مخصص بشبكة سريعة (تقليل المهلة الزمنية):Bashpython async_recon.py -t 127.0.0.1 -to 0.5 -c 2000 -s 1 -e 1024
