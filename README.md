# AsyncRecon Framework

إطار عمل متقدم للفحص الشبكي غير المتزامن (Asynchronous Port Scanning) مصمم لاكتشاف المنافذ المفتوحة واستخراج ترويسات الخدمات (Banner Grabbing) بسرعات فائقة.

## 🚀 لماذا AsyncRecon؟
في بيئات اختبار الاختراق، السرعة هي المفتاح. تعتمد الأداة على تقنيات غير متزامنة متطورة لتقليص وقت الاستطلاع من دقائق إلى ثوانٍ معدودة.

### الميزات التقنية:
* **محرك فائق السرعة:** يعتمد على `uvloop` (البديل الأسرع لـ `asyncio` القياسي).
* **تحسين الموارد:** إدارة ذكية لواصفات الملفات (File Descriptors) لمنع فشل الاتصالات في بيئات التزامن العالي.
* **واجهة تفاعلية (TUI):** استخدام مكتبة `rich` لعرض حالة الفحص، شريط التقدم، والنتائج الحية بشكل احترافي.
* **دعم Banner Grabbing:** استخراج أولي للترويسات للتعرف على نوع الخدمة العاملة على المنفذ.

---

## 🛠 المتطلبات (Requirements)
* Python 3.8+
* نظام تشغيل UNIX-like (Linux/macOS) - للحصول على أفضل أداء للـ `uvloop`.
